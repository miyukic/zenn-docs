## 国産LLMは今どこにあるか

2026年4月時点で、ツイートで触れられていた三社の状況はこんな感じ。

**PLaMo（プリファード・ネットワークス）**
三社の中で一番積極的に公開してる。Hugging Faceにモデルウェイトが置いてあるし、[plamo.preferredai.jp](https://plamo.preferredai.jp/) でWebチャットとOpenAI互換APIが無料で使える。今すぐブラウザから試せる。全国150以上の自治体が使うQommonsAIに標準搭載されてたりと、実績もちゃんとある。

**Sarashina（ソフトバンク / SB Intuitions）**
Hugging FaceにSarashina2シリーズ（7B〜70B）がMITライセンスで公開されてる。商用利用可。GGUFもあるのでOllamaでローカル実行もできる。

**cotomi（NEC）**
これは非公開。ウェイトの一般公開はなく、企業・行政向けにAPIとオンプレミスで提供される。デジタル庁の「源内」プロジェクトで試用モデルに選定された実績はあるけど、個人が触る手段はない。

つまり2社はちゃんと公開してる。触ろうと思えば動かせるPCさえあれば誰でも今日から触ることができる！
結構熱い！！

:::message
国産LLMはこの三社以外にもNTTのtsuzumiや楽天AI、国立情報学研究所のLLM-jpなど複数存在する。この記事では元ツイート関連で触れられていた三社を中心に扱う。
:::

## 日本が繰り返してきたパターン

これは国産LLMだけではなくて・・・。

PCゲームを思い出してほしい。ValveがSteamを作り、世界中の開発者が集まってPCゲーム市場が育っていった2010年代、日本の大手ゲームメーカーは「PCゲームは割れる」「コンシューマで十分」と言い続けた。

フリーゲームやMOD文化は日本にも昔からあった。一部PCユーザーの間ではPCでゲームをしたい！という声は当時からあった。

でも商用タイトルをPCに出すという選択は長らく忌避されてたように思う。

海外勢がPCゲーム市場を作ってやっと日本もPCでゲームを出すようになったように思う

ラズパイも同じだ。ホビー向け低消費電力マイコンという市場の隙間を、日本メーカーは埋めようとしなかった。その隙間にイギリスのRaspberry Pi財団が入り込んで、世界の開発者教育インフラになった。

ロボットもそうだ。ソニーはAIBOでカワイイおもちゃのポジションを選んだ。本格的な汎用ロボットはBtoBでコソコソやってる間に、Boston Dynamicsがバズって、安価版がAliExpressに溢れた。

**「最初から売れないと決めつけて、売ろうともしない」**

これが繰り返されてるパターンだ。売らないから売れない。当たり前の話だ。

## 技術は使われてこそ進歩する

ここが一番言いたいことかもしれない。

OpenAIがChatGPTを一般公開したとき、ハルシネーションだらけだった。それでもとりあえず出した結果、何億人ものユーザーが使い、フィードバックが爆発的に集まって、想定外のユースケースが無数に発見された。出すこと自体がR&Dになった。

国産LLMは研究所とBtoB契約先にしか届いてない。フィードバックループが細すぎる。

日本語の敬語・公文書・業界用語で強いという特化型の強みは、大衆に使い倒されてフィードバックを受けることで初めて本当に育つ。行政の限られたユースケースだけじゃ、鍛えられる範囲が狭すぎる。

"誰でも簡単に使えるようにする"じゃなくて、"やる気があれば触れる環境を作る"こと。前者は天井を下げる。後者は可能性を開く。

そっちを目指してほしいと思う。

## 今すぐ試す方法

とはいえ公開されてるものは誰でも今日から触れる。
ぜひ、国産LLMを試してほしい。

**PLaMoを試す**
[plamo.preferredai.jp](https://plamo.preferredai.jp/) でWebチャットが無料で使える。
OpenAI互換APIも提供されてるのでコードから呼ぶこともできる。まず触ってみるならここが一番敷居が低い。

**Sarashinaをローカルで動かす**
Hugging Faceに小型モデルがある。GGUFがあるのでOllamaで動く。

```bash
ollama run hf.co/mmnga/sarashina2.2-3b-instruct-v0.1-gguf
```

公式はGGUFを出していないため、コミュニティが変換・公開したモデルを使う形になる。日本語特化の感触をローカルで試せる。
